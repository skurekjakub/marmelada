    Reference - Web parts

    This page lists the web parts, in alphabetical order, that the system offers out-of-the-box.

    You can find examples demonstrating how you can use the individual web parts in the Examples -> Web parts section of the sample Corporate site.

    License limitations

    Based on your Kentico license and the permissions granted, you may not be able to use some of the web parts.

    Further web part information

    This page only provides basic information about web parts. To find more information, search our documentation or configure one of the web parts in the Pages application and click the help icon in the top right corner of the configuration dialog.

    Related pages

    • Using and configuring web parts

    • Preparing widgets for users

    Web part

    Description

    Abuse report

    Allows users to write and submit reports concerning website abuse.

    Abuse report list

    Displays a list of abuse reports submitted by website users and allows them to be managed.

    Available only as a widget.

    Accordion layout

    This web part provides an accordion layout divided into multiple panes that can be expanded or collapsed. Each pane defines a separate web part zone, so any content may be added into the layout. Users can click on the pane headers to select which one should be displayed.

    Activities

    This web part shows a list of activities, i.e. actions such as visiting a page, making a purchase, subscribing to an event etc., performed by a contact on the website.

    Available only as a widget.

    Admin actions

    Displays a link to the administration interface, and a link that opens the current page for editing. You can set the text of the links a select which links are visible.

    Analytics browser capabilities

    Collects information about the browser and system used by visitors that access the page. The results can be viewed in the Web analytics application under the Browser capabilities category.

    Analytics custom statistics

    This web part logs a hit for a custom web analytics statistic when visitors access the page.

    Atom data source

    Creates data source from Atom feed which is specified by URL. Result contains columns depending on given Atom feed.

    Atom feed

    The Atom feed web part can be used to create an Atom feed. It must be connected to a data source web part, which will provide data for the feed. It displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Atom repeater

    This web part changes a page to an Atom feed of data provided by a connected data source web part.

    Attachments

    Displays a list of CMS.File documents that are located under the document containing the web part in the content tree. Files are displayed based on the assigned transformation.

    Attachments data source

    Creates a data source containing the attachments of a specified page. This data source may be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Automatically initiated chat

    This web part allows you to automatically contact site visitors with predefined messages after a specified period of time.

    Banner rotator

    Displays banners from a specified banner category according to their weight.

    Basic Bing maps

    The web part displays maps obtained from the Bing maps service. This web part must be used together with a data source web part.

    Basic datalist

    The Basic datalist web part displays a multicolumn/multirow list of specified objects based on assigned transformations. This web part must be used together with a data source web part.

    Basic Google maps

    The web part displays maps obtained from the Google maps service. This web part must be used together with a data source web part.

    Basic repeater

    The Basic repeater web part renders specified objects using the assigned transformations. This web part must be used together with a data source web part.

    Basic repeater with effect

    Displays data objects provided by a data source web part using the assigned transformations. The web part can additionally be used to apply a JavaScript effect to the items. The scripts used to generate the effect must be specified through the web part's properties.

    Basic universal viewer

    The Basic Universal Viewer web part renders specified objects using assigned transformations. It is able to display data from hierarchical grouped data sources. This web part must be used together with a data source web part.

    Bing maps

    A page based web part which displays maps obtained from the Bing maps service.

    Bing translator

    Displays the Microsoft Bing translator box which allows users to translate the current page into a variety of languages.

    Blog comments

    Displays a list of comments that were added to the specified blog(s). Authorized users will also be able to approve, reject or edit the listed comments.

    Available only as a widget.

    Blog comments data source

    Creates a data source containing comments added by users to blog posts. This data may then be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Blog comments RSS feed

    This web part creates an RSS feed of comments loaded from a Kentico blog. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Blog post subscription confirmation

    Approves subscriptions to blog posts with double opt-in enabled and displays a confirmation message. Subscribers are typically directed to the page containing this web part through a link in the double opt-in activation email.

    Blog post unsubscription

    Displays a confirmation that the user has been successfully unsubscribed from the blog posts board. The subscription is specified by the GUID parameter in the URL.

    Blogs comments viewer

    Displays blog comments based on the filter settings in the web part properties.

    Board messages data source

    Retrieves the content of message boards and provides it as a data source. This data can then be connected to a listing web part (such as a Basic repeater) in order to be displayed.

    Breadcrumbs

    (available in the Navigation web part category)

    The Breadcrumbs web part allows users to see their current position within the structure of the website. Also provides links to all ancestor documents (previous levels).

    Calendar

    (available in the Listing and pages -> Pages web part category)

    Displays a calendar containing data from specified pages based on assigned transformations.

    Category breadcrumbs

    Shows breadcrumb navigation inside the hierarchical categories.

    Category list

    Displays the list of specified categories.

    Category menu

    Tree menu built from the specified categories.

    Change password

    This web part allows users to change their own password.

    Chat errors

    Displays errors, warnings and alerts (such as wrong password, flooding etc.).

    Chat leave room

    Displays a button for leaving a chat room.

    Chat login

    Enables users to log into chat. When logged in, displays the login status, a link to change the nickname and the Leave button.

    Chat notifications

    Displays notification status area, informing users how many notifications (e.g., room invitations) they have and allows to open the list of notifications.

    Chat on-line users

    Displays a list of all users currently on-line in the chat.

    Chat room messages

    Displays messages in chat room.

    Chat room name

    Displays the name of the room the user is currently in.

    Chat room users

    Displays the list of users in a chat room.

    Chat rooms

    Displays the list of chat rooms.

    Chat search on-line users

    Allows to search in users currently on-line in the chat.

    Chat send message

    Allows users to send messages to a chat room.

    Chat support request

    Displays a button that users can click to request on-line support.

    Chat web part

    Easy to use all-in-one web part compiling all the chat features into a single component.

    Checkout process (obsolete)

    Checkout process wizard. Allows users to purchase items from their shopping cart. This web part incorporates all steps of a usual checkout process. If you want to customize the checkout process, use individual checkout process web parts instead of this one.

    CMS RSS feed

    Creates an RSS feed of Kentico pages for specified page types. Uses a built-in pages data source as a source of the feed data. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Collapsible panel

    Displays a panel that can be collapsed or expanded by users. The panel contains a web part zone, so any type of content may be added.

    Columns layout

    Provides a CSS-based column layout for content. Each column defines a separate web part zone, so any type of content may be added.

    Comment view

    Displays all comments of the specified blog post.

    Confirmation checkbox

    This web part can be placed on a wizard layout web part and the user has to check the checkbox in order to continue. It can be used for example during checkout when the user needs to agree to some terms in order to continue in the checkout.

    Contact list

    The Contact list web part allows users to manage their contact list.

    Content rating

    Used to enable rating of page content.

    Content slider

    Displays a slideshow of the content of specified pages based on the assigned transformations.

    Content subscription

    Using this web part any user can subscribe to the email notifications about page events (insert, update, delete).

    Contribution list

    Displays a list of contributions or only the New page button.

    Cookie law and tracking consent

    Provides a dialog to get user consent with cookie usage and tracking of personal data. Allows you to display particular message and actions for particular cookie levels and selected consent object.

    Coupon codes

    Allows customers to enter coupon codes during the checkout process to receive discounts or redeem gift cards.

    See also: Working with coupon codes

    CSS list menu

    Renders a navigation menu for a specified section of the website using <UL> and <LI> elements. It may utilize advanced CSS styles to display any type of menu, such as a drop-down menu. It produces much less code than other types of menus, but requires advanced knowledge of CSS styles and positioning to render advanced menu behavior.

    CSS style selector

    Generates links that allow users to select up to five different styles for the website according to their preferences. The selected settings are stored in a browser cookie.

    CSS stylesheet

    Adds a request for an external CSS stylesheet file to the HEAD element of the page.

    Currency selection

    Allows users to select a currency that E-commerce web parts use for displaying prices. Place the web part either on a checkout process page or anywhere on your website, e.g., on your master page header. Note that if there is only one valid currency in your store, the Currency selection is automatically hidden. The web part only offers currencies with valid exchange rate.

    See more in Reference - Checkout process web parts.

    Current user

    Displays the user name and full name of the current user.

    Custom registration form

    Allows visitors to register on the website as new users. The displayed registration interface is based on an alternative form.

    Custom response

    Adding this web part to a page causes it to return a custom HTTP response when the page is requested. The response is only generated on the live site.

    Custom subscription form

    Displays a customizable dialog that allows visitors to subscribe to the website's newsletters. The subscription dialog is based on an alternative form.

    Custom table data source

    Loads data from a specified custom table and provides it as a data source, which can then be displayed by listing web parts (such as a Basic repeater).

    Custom table datagrid

    Displays a grid (table) containing data retrieved from a specified custom table.

    Custom table datalist

    Displays a multicolumn/multirow list containing data retrieved from a specified custom table. The format of the data is determined by assigned transformations.

    Custom table form

    Allows you to insert and edit custom table items directly on the page.

    Custom table repeater

    Renders data retrieved from a specified custom table based on assigned transformations.

    Custom table repeater with effect

    Renders data retrieved from a specified custom table according to the assigned transformations and applies an additional javascript effect to the displayed records. The scripts used to generate the effect must be specified through the web part's properties.

    Custom table RSS feed

    This web part creates an RSS feed of data stored in Kentico custom tables. Uses a built-in custom table data source as a source of the feed data. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Customer address

    Allows customers to enter their billing and/or shipping address during the checkout process.

    See more in Reference - Checkout process web parts.

    Customer detail

    Allows customers to enter their personal and business information during the checkout process.

    See more in Reference - Checkout process web parts.

    Customer edit address form

    Allows customers to create or edit their addresses.

    Customer edit company details

    Allows customers to edit their company details.

    Datalist

    Displays the content of specified pages in a multicolumn/multirow list based on assigned transformation.

    Datalist for web service

    Displays a multicolumn/multirow list of data provided through a web service. The data is formatted using a specified transformation.

    Datalist with custom query

    Displays a multicolumn/multirow list containing data retrieved by a specified query. Data is formatted according to the assigned transformations.

    Date & time

    Displays the current date and time.

    Department members data source

    Creates a data source containing users who are members of a specified department (users who have the Read permission for the department's main page are returned). This data may then be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Department members viewer

    Displays a list of users who are members of a specified department. Displayed users can be filtered using the Filter section of web part properties.

    Did you mean

    Checks for spelling errors in search expressions and offers a link that searches for the modified search terms. It can be used together with any of the Smart or SQL search web parts.

    Document pager

    Displays a single item at a time from a set of specified pages based on the assigned transformation. A pager is used to switch between items.

    Edit contribution

    Enables editing of user contributions.

    Edit page link

    Displays a link that opens the current page for editing (in the Pages application or in on-site editing mode). You can configure the web part to display the link only for users with permissions to edit the page.

    Editable image

    Allows content editors to select a single image on the Page tab of the Pages application, or when viewing the page in On-site editing mode. The web part displays the selected image on the live site.

    Editable text

    Provides an editable text region that can be used to enter page content. Users can write content on the Page tab of the Pages application, or when viewing pages in On-site editing mode.

    Email feed unsubscription

    Processes newsletter and email campaign unsubscription requests submitted by recipients and displays a confirmation message.

    Email queue

    Displays a list of emails that are currently waiting to be sent and those that were not sent successfully. It also provides an interface for resending emails.

    Available only as a widget.

    Event calendar

    Displays in calendar scheduled events.

    Event management

    Allows manage events, modify their attendees and email notifications.

    Available only as a widget.

    Event registration

    Displays list of booked events.

    EventLog

    Displays the system event log. You can filter which events should be included in the log. Detailed information about each of the logged events can also be viewed.

    Available only as a widget.

    Events RSS feed

    This web part creates an RSS feed of Kentico events. Uses a built-in pre-configured pages data source as a source of the events. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Facebook Connect logon

    This web part allows users to log into the website using their Facebook accounts.

    Feed link

    The Feed link web part displays a link to RSS, Atom or XML feed based on the web part settings. It supports automatic feed discovery.

    File system data source

    Retrieves data from files stored in the local file system, which may be provided to listing web parts (such as a Basic repeater). The list of files is generated according to a specified directory path and search pattern.

    Filter

    This web part is used to place custom developed filters onto your website and connect them to a datasource. The path to the user control implementing the filter must be specified.

    Forum - Most active threads

    Displays a list of the most active threads from the selected forums based on the specified transformation. The order of the threads is determined by their post count.

    Forum - Recently active threads

    Displays a list of forum threads that were most recently active (i.e. new posts were added to them). Threads are loaded from the selected forums and their format is based on the specified transformation.

    Forum - Top contributors

    Displays a list of users who are the most active forum contributors, formatted according to a specified transformation. The order of the users is determined by their amount of posts in the forums on the current website.

    Forum (Single forum - Flat layout)

    Displays the content of a specified forum in a flat, thread-based layout. This web part is obsolete. We recommend using the Forum (Single forum - General) web part instead and selecting the Flat layout.

    Forum (Single forum - General)

    Displays the posts contained in a specified forum. Users may browse the content of this forum, add posts and perform other actions.

    Forum (Single forum - Tree layout)

    Displays the content of a specified forum using a hierarchical tree layout. This web part is obsolete. We recommend using the Forum (Single forum - General) web part instead and selecting the Tree layout.

    Forum breadcrumbs

    Displays breadcrumb navigation for forums on the same page, which allows users to see their current location in the forums and link to previous levels. This web part is helpful if the layout of the forum doesn't display breadcrumbs by default.

    Forum favorites

    Displays a list of the favorite forum items of the current user. The items serve as links to the given forum section.

    Forum group

    Displays the forums contained in a specified forum group. Users may browse the content of these forums, add posts and perform other actions.

    Forum posts data source

    Retrieves posts from the specified forum(s) and provides them as a data source. This data must be connected to a listing web part (such as a Basic repeater) in order to be displayed.

    Forum posts RSS feed

    This web part creates an RSS feed of Kentico forum posts. Uses a built-in forum posts data source as a source of the feed data. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Forum posts viewer

    Displays the content of posts from the selected forums based on the specified transformations.

    Forum posts waiting for approval

    Displays posts from moderated forums that are waiting for approval. Only posts that the current user is allowed to approve are shown.

    Available only as a widget.

    Forum search - advanced dialog

    Displays an extended forum search dialog that allows users to search for keywords in the content of forums. Several parameters can be set that specify the scope of the search. When a search request is submitted, the web part redirects to a page where the results are displayed. This web part uses the SQL search engine.

    Forum search box

    Displays a simple search box that allows users to look for a specified expressions in the content of forums. When a search request is submitted, the web part redirects to a page where the results are displayed. This web part uses the SQL search engine.

    Forum search results

    Displays the results of search requests entered into Forum search box or Forum search - advanced dialog web parts.

    Forum subscription confirmation

    Approves subscriptions to forums or forum posts with double opt-in enabled and displays a confirmation message. Subscribers are typically directed to the page containing this web part through a link in the double opt-in activation email.

    Forum unsubscription

    This web part processes forum unsubscription requests submitted by users and displays a confirmation message.

    Friends data source

    Creates data source from friends of given user. You can filter results by e.g. appropriate friend status or site name.

    Friends list

    Shows list of friends.

    Friends viewer

    Displays friends using selected transformation.

    Friends waiting for approval

    Shows list of friends waiting for approval.

    Friendship management

    Allows to manage friendship through given GUID.

    Google analytics

    Includes tracking code for Google analytics service.

    Google maps

    A page based web part which displays maps obtained from the Google maps service.

    Google search

    Displays a Google search box with results depending on settings made at the Google Custom Search website (http://www.google.com/cse/).

    Google Sitemap (XML Sitemap)

    Generates an XML sitemap or sitemap index according to the Google Sitemap protocol. The resulting page can then be processed by search engine web crawlers (robots).

    Google translator

    Displays the Google translator box which allows users to translate the current page into a variety of languages.

    Grid

    (available in the Listings and viewers -> Pages web part category)

    Displays a grid (table) containing data fields of specified pages or a detail of a selected page based on an assigned transformation.

    Grid for REST Service

    Displays data retrieved using the Kentico REST Service in a table.

    Grid for web service

    Displays a grid (table) containing data retrieved from a specified web service.

    Grid with custom query

    Displays a grid (table) containing data retrieved by a specified query.

    Group contribution list

    Displays a list of group's contributions or only the New page button.

    Group edit contribution

    Enables editing of user contributions.

    Group forum - Most active threads

    Displays a list of the most active threads from the forums that belong under the selected group. The forum threads are displayed based on the specified transformation and their order is determined by post count.

    Group forum - Recently active threads

    Displays a list of threads from the forums that belong under the selected group that were most recently active (i.e. new posts were added to them). The format of the listed threads is based on the specified transformation.

    Group forum list

    Displays all forums belonging under the selected community group (or workgroup). Users may browse the content of these forums, add posts and perform other actions.

    Group forum posts data source

    Retrieves posts from forums that belong under a specified group and provides them as a data source. This data must be connected to a listing web part (such as a Basic repeater) in order to be displayed.

    Group forum posts viewer

    Displays the content of posts from the forums that belong under the selected group based on the specified transformations.

    Group forum search results

    Displays the results of search requests entered into Forum search box or Forum search - advanced dialog web parts. Only the content of the forums belonging under the currently active group will be searched.

    Group forums

    Allows group administrators to manage the forums of the specified group directly from the live site.

    Group invitation

    Manages invitations to join the group. The GUID of the invitation that should be managed must be passed to the web part through the URL via the 'invitationguid' query string parameter.

    Group media libraries

    Allows group administrators to manage the media libraries of the specified group directly from the live site.

    Group media libraries data source

    Creates a data source containing media libraries that belong under a specified group. This data can then be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Group media libraries viewer

    Used for displaying group media libraries.

    Group members

    Allows group administrators to manage the members of the specified group directly from the live site.

    Group members data source

    Creates a data source containing users who are members of a specified group. This data may then be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Group members viewer

    Displays a list of users who are members of the specified group based on the specified transformations. It is possible to limit which users should be loaded using the web part's properties.

    Group message board

    Displays a message board, including a list of all posted messages and a dialog for adding new messages. The board will belong under the current group.

    Group message board viewer

    Used for displaying group message boards.

    Group message boards

    Allows group administrators to manage the message boards of the specified group directly from the live site.

    Group permissions

    Allows group administrators to manage the page permissions of the specified group directly from the live site.

    Group poll

    The Poll web part allows you to insert your own Poll on the page.

    Available only as a widget.

    Group polls

    Allows group administrators to manage the polls belonging under the specified group directly from the live site.

    Group profile

    Allows group administrators to manage the profile, properties and content of the specified group directly from the live site.

    Group properties

    Allows group administrators to edit the basic properties of the specified group directly from the live site.

    Group public profile

    Displays the public profile of the specified group. The layout and content of the profile is defined by a selected form.

    Group registration

    Allows users to create new groups directly from the live site.

    Group roles

    Allows group administrators to manage the roles of the specified group directly from the live site.

    Group security access

    This web part checks if the current user has sufficient permissions to view the content of the group to which the given page belongs. Unauthorized users are automatically redirected to a specified URL.

    Group security message

    Displays a message to users who do not have sufficient permissions to view the content of the currently active group. A different message can be displayed to different types of users.

    Groups data source

    Creates a data source containing groups selected according to the specified properties. This data may then be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Groups filter

    Displays a filter that can be used to limit which groups should be displayed from the connected data source. It is intended for use in combination with a Groups data source web part.

    Groups viewer

    Displays a list of groups based on the specified transformations. It is possible to limit which groups should be loaded using the web part's properties.

    Head HTML code

    This web part inserts custom HTML code into the HEAD element of the page.

    Hierarchical viewer

    The Hierarchical viewer is based on the Universal viewer and contains only those properties necessary for displaying hierarchical data. The web part renders the content of specified pages using the assigned transformations.

    Ignore list

    The Ignore list web part allows users to manage their ignore list. Owner of the ignore list don't recieve message from users in the ignore list.

    Image gallery

    Displays a multi-column image gallery with the option of viewing thumbnails or full-size image details. The images are loaded from specified CMS.File documents and rendered based on assigned transformations.

    Inbox

    The Inbox web part displays recieved messages for current user.

    Initiated chat

    This web part allows visitors to respond to chat conversation initiated by you from the On-line users or On-line contacs page.

    In-line abuse report

    Displays a link that opens a modal dialog used for submitting abuse reports.

    Javascript

    This web part inserts javascript onto the page. The script can either be defined in-line or retrieved from an external file.

    Keep alive

    This web part generates hidden requests for the given page to prevent the current user's session from expiring. The time interval between requests can be specified.

    Language data source

    Creates a collection of culture specific URLs. The available fields are 'CultureCode', 'CultureName', and 'URL'. You can pair the web part with a Basic repeater to create a language selector.

    You can see this web part used on the sample Dancing Goat website for reference.

    Leave group

    Allows to users to leave the currently active group (cancel their membership). The current group is taken from the context, which means that the web part must be placed onto a page that is owned by a group in order to work correctly.

    Link / Button

    Displays a hyperlink to a specified target URL. The link can be rendered either in text format or as an image.

    LinkedIn logon

    This web part allows users to log into the website using their LinkedIn accounts.

    LinkedIn required data

    Allows users to merge their LinkedIn accounts with their existing account on the website. It may also be used to request additional information from new users who register through their LinkedIn account.

    LiveID required data

    Allows users to merge their LiveID accounts with their existing account on the website. It may also be used to request additional information from new users who register through their LiveID account.

    Logon form

    Displays a form that allows users to log into the website. Authentication requires a valid user name and password.

    Logon mini form

    Displays a simplified form that allows users to log into the website. Authentication requires a valid user name and password.

    Macro data source

    Retrieves data using a macro expression and provides it as a data source. To display the data, connect this data source to a listing web part (such as a Basic repeater).

    Media file uploader

    Using this web part you can upload files to the specified media library.

    Media files data source

    Creates a data source containing files stored in a specified media library. This data can then be connected to listing web parts (such as a Basic repeater) in order to be displayed.

    Media files RSS feed

    This web part creates an RSS feed of files stored in Kentico media libraries. Uses a built-in media file data source as a source of the feed data. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Media gallery

    Web part used for displaying content of a media library.

    Media gallery - file filter

    Enables sorting files in media gallery.

    Media gallery - file list

    Displays files in the selected folder in a media library and allows playing or opening the files.

    Media gallery - folder tree

    Displays folder tree structure of the media library and enables user to browse it.

    Media libraries data source

    Creates a data source containing media libraries, which can then be provided to listing web parts (such as a Basic repeater) in order to be displayed.

    Media libraries viewer

    Displays list of media libraries.

    Message board

    Displays a message board, including a list of all posted messages and a dialog for adding new messages. The board can either be public or owned by a specified user.

    Message board RSS feed

    This web part create an RSS feed of message board messages. Uses a built-in board messages data source as a source of the feed data. Displays a link leading to the feed, which is located at the URL of the page where the web part is placed, with a querystring parameter appended.

    Message board subscription confirmation

    Approves subscriptions to message boards with double opt-in enabled and displays a confirmation message. Subscribers are typically directed to the page containing this web part through a link in the double opt-in activation email.

    Message board unsubscription

    Displays a confirmation that the user has been successfully unsubscribed from the message board. The subscription is specified by the GUID or hash parameter in the URL.

    Message board viewer

    Displays the list of messages from all message boards on the specified site according to the given WHERE condition.

    Message list

    Displays a list of posts that were added to the specified message board(s). Authorized users will also be able to approve, reject or edit the listed posts.

    Available only as a widget.

    Message panel

    Displays error messages generated during a multi-page checkout process.

    See more in Reference - Checkout process web parts.

    Messaging info panel

    Messaging module info panel with links to Inbox, Outbox and New message.

    Mobile device redirection

    Redirects mobile device visitors accessing the page where it is placed to a specified URL. Two different redirection URLs can be specified based on if the device is recognized as a large or a small device. Its typical use is when creating a dedicated website section for mobile devices. It is typically placed on the website's main Home page and redirects mobile users to the Home page of the dedicated mobile section.

    My account

    Displays various information about the current user's account and allows its settings to be managed. This web part is only visible for authenticated users.

    My accounts

    This web part shows list of accounts owned by current user.

    Available only as a widget.

    My contacts

    This web part shows list of contacts owned by current user.

    Available only as a widget.

    My friends

    Displays information about user friends.

    My invitations

    Displays a list of invitations to join a group for the specified user. Users can accept or delete their invitations.

    My messages

    The My messages web part provides user the possibility of managing recieved and sent messages. User can also manage his contact and ignore list.

    My newsletter subscriptions
