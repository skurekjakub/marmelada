    Using banners - example

    This example will show you the whole process of banner management, from creating a banner category and filling it with banners, to displaying the banners on a site. The example uses the sample Corporate site.

    Creating a banner category

    1. Open the Banners application.

    2. In the Site selector, select the Corporate site.

    3. Click New banner category.

    4. Set the banner category properties as follows:

      • Display name: Header banners

      • Enabled: Selected

    5. Click Save.

    Creating individual banners

    First banner

    1. While still in the banner category that you created, click New banner.

    2. Set the new banner's properties as follows:

      • Display name: Banner 1

      • URL: http://www.kentico.com

      • Weight: 1

      • Open in new window: Selected

      • Enabled: Selected

      • Banner type: Plain text

      • Banner content: Any text you like

    3. Click Save.

    Second banner

    1. Return to the list of banners on the Banners tab, click New banner again.

    2. Set the new banner's properties as follows:

      • Display name: Banner 2

      • URL: http://devnet.kentico.com

      • Weight: 4

      • Open in new window: Selected

      • Enabled: Selected

      • Banner type: HTML

      • Banner content: Any formatted text you like

    3. Click Save.

    Third banner

    1. Return to the list of banners on the Banners tab, click New banner again.

    2. Set the new banner's properties as follows:

      • Display name: Banner 3

      • Code name: banner3

      • URL: http://www.kentico.com/Download-Demo

      • Weight: 5

      • Open in new window: Selected

      • Enabled: Selected

      • Banner type: Image

    3. Click Select next to the Banner image property.

    4. On the Content tab, upload a file of your choice into the Images folder and click Select.

    5. Fill in the Image title and Alternative text fields.

    6. Click Save.

    Placing banners on a page

    You have created a set of banners that will rotate in the header of the Corporate site. Now place the Banner rotator widget on the Corporate site's Home page as described in the following steps:

    1. Open the Pages application on the Page tab.

    2. Select the Home page in the page tree.

    3. Open any of the page's zone menus ( ).

    4. Click Add new widget.


      Placing banner on a page

    5. Select the Banner rotator widget.

    6. Click Select.

    7. Set the widget's Banner category, Width, and height properties.

      Configuring banner properties

    8. Click Save & Close.

      • The widget appears within the content of the page (inside the selected zone), but is NOT permanently saved yet.

    9. While still on the Page tab, click Save to confirm the changes to the page's widget content.

    The result

    You have created three sample banners and configured the Banner rotator web part to display them to visitors on the Corporate site.

    When you switch to the live site, you will be presented with one of the three banners. When you refresh the page several times, the other banners will appear with the frequency proportional to their Weight. For example, out of 100 page views, the banners' displaying frequency will be as follows:

    • Banner 1 has a weight of 1, therefore it will be displayed approximately 10 times.

    • Banner 2 has a weight of 4, therefore it will be displayed approximately 40 times.

    • Banner 3 has a weight of 5, therefore it will be displayed approximately 50 times.
