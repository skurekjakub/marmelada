    Creating campaigns

    OBSOLETE

    Use Managing campaigns

    On-line marketing campaigns allow you to track and evaluate your efforts in promoting special targets on your website.

    For example, you can create a campaign which would track how many of your clients download a whitepaper from a specific page on your website after you send the clients a promotional email with a link to that page.

    You can see an overview of your on-line marketing campaigns in the Campaigns application. Here you can see a list of all your campaigns with their statuses and details.

    Creating a new campaign

    To create a new campaign:

    1. Open the Campaigns application.

    2. Click Create new campaign.

    3. Type the Display name of the new campaign.

      • The UTM campaign field is filled automatically according to the campaign name. The field defines the name of the campaign used in the utm_campaign parameter in URLs of links. For example for MyCampaign, the URLs will have the following form: http://www.mywebsite.com/Home.aspx? utm_campaign =MyCampaign.

    4. Type the Description of the campaign - information about its purpose, goals and so on.

    Campaigns are saved automatically.

    Adding promotional emails to the campaign

    Promotional channels allow you to define sources, through which you will address your target audience. In Kentico, you can define emails as the sources for campaign promotion. When you create a promotional email and associate it with a campaign, the system will recognize visitors that come to your website by clicking the links in the email.

    To associate emails with a campaign:

    1. Click Add campaign email.

    2. Select an existing email which you want to associate with your campaign.

    The system associates the email with the campaign and displays it in the Campaign promotion section.

    The UTM Source field defines the value for the utm_source parameter in the URL of links. The system automatically includes the parameter with the defined value into all links in the selected email.

    Defining other promotional sources

    However, you can also define other sources for campaign promotion, for example through Facebook or Twitter. You only need to compose a correct URL which you will include in your posts. When visitors click on links containing such URL, the system will recognize the social network which they came from and will log appropriate statistical data for them.

    Defining campaign content

    The campaign content defines a path of the visitor to the target of the campaign. You can add different pages, files or forms to the campaign. The system then tracks the page visits, file downloads and form submissions for the added items. For example, if the target of the campaign was to download a whitepaper, you would add a page on which the link for downloading the whitepaper is located, and a file download for the particular whitepaper.

    Specify the Conversion name and Conversion value fields for individual page visits. The Conversion name defines the name which will be displayed on the Report tab. You can define the same conversion name for multiple page visits.

    Note: You can display the campaigns a page is a part of in Pages (application) -> Analytics -> Settings.

    Order of campaign content items

    Remember to define the campaign content in a logical order in which the visitor performs the desired actions. For example 1. Whitepapers page visit, 2. Whitepaper file download.

    Only when configuring the correct order of actions will the system be able to display campaign data and statistics correctly.

    Creating new pages from the Campaign interface

    The default location for new pages created in the new campaign interface is defined by the New page location setting in the Settings application -> On-line marketing.

    Launching a campaign

    When you have defined your campaign, check if you have completed all items on the launch checklist we provided and click Launch campaign. The system sends unsent and unscheduled campaign emails and publishes unpublished pages (that are not under workflow and are not scheduled to be published) upon launching the campaign.

    Finishing and evaluating a campaign

    When you no longer want to track the actions visitors perform through this campaign, click Finish campaign. The system will stop tracking the visitors. You cannot run a campaign again once it was closed.

    Switch to the Report tab to view the campaign data and graphs from which you can evaluate the campaign.
